Luma v2

Advanced, Scalable Discord Bot

Luma v2 is a next-generation Discord bot designed for enterprise-level communities. With a dynamic plugin system, multilingual support, and deep customization options, Luma powers servers at scale.

⭐ Key Features

  • Dynamic plugin system with hot-reloading
  • Multilingual support (i18n) for global communities
  • 5 command categories: Community, Fun, Management, Moderation, Utility
  • Intelligent caching system with configurable timeouts
  • PostgreSQL database with asyncpg for reliability
  • Advanced dashboard with interactive games
  • Automatic slash command synchronization
  • Ticket system, mail system, role panels, and more

🎮 Dashboard & Gaming

The Luma dashboard isn't just for configuration — it includes an interactive game environment where server owners can engage with their community in new ways. The dashboard provides analytics, configuration management, and gaming integrations all in one place.

💡 Perfect For

  • Large communities requiring scalability
  • Multilingual servers and international communities
  • Servers needing custom, specialized features
  • Advanced server administration and automation
  • Communities wanting an interactive gaming experience

🔧 Tech Stack

  • Python 3.10+
  • discord.py 2.4+
  • asyncpg (PostgreSQL async driver)
  • aiohttp for async requests
  • Railway deployment ready
  • Database migrations system

🏆 Why Choose Luma?

  • ✅ Modular architecture — easy to extend and maintain
  • ✅ Production-ready with active maintenance
  • ✅ Performance optimized with intelligent caching
  • ✅ Native multilingual support built-in
  • ✅ Interactive dashboard with games and analytics
  • ✅ 24/7 uptime focus with reliability practices

KRM - Kurayami

Specialized Welcome & Exit Bot

KRM (Kurayami) is a specialized Discord bot focused on creating welcoming experiences for new members and personalized farewells. Inspired by Japanese Oni mythology, it represents strength, unity, and community presence.

⭐ Key Features

  • Automated welcome messages for new members
  • Personalized exit/goodbye messages
  • Interactive configuration via slash commands
  • Select menu-based interface for better UX
  • Message preview and testing before deployment
  • Birthday celebration system
  • Member join/exit tracking
  • Persistent storage in PostgreSQL

🎯 Perfect For

  • Communities wanting personalized member experiences
  • Servers that need dynamic, welcoming messages
  • Automatic member tracking and anniversaries
  • Lightweight bots without unnecessary overhead
  • Quick setup and deployment

🔧 Tech Stack

  • Python 3.10+
  • discord.py 2.4
  • asyncpg (PostgreSQL async)
  • python-dotenv for configuration
  • Docker support included

🏆 Why Choose KRM?

  • ✅ Simple to set up and configure
  • ✅ Intuitive interface with select menus
  • ✅ Lightning-fast performance
  • ✅ Reliable data persistence
  • ✅ Perfect for specific, focused use cases
  • ✅ Docker-ready for easy deployment
  • ✅ Lightweight with minimal resource usage

📚 Mythology & Inspiration

The name "Kurayami" (闇 - darkness) draws inspiration from Japanese Oni mythology. Like the entities they represent, KRM serves as a symbol of presence, strength, and community unity. Founded in the shadows, guided by leadership and respect, the KRM foundation represents the beginning of a legacy built with honor and presence.
